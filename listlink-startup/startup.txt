Read and internalize this blurb about what the website mylistlink.com will be:
Mylistlink is where curious and interesting people go to create, share, and learn. Each user makes lists of 7 items around a theme. An example is “Best icecream flavors”. Users explore the lists of others via search and, more excitingly, via “crosses” where a user’s list may contain 1 or more of the same elements as another user. Using the above example, another user may make a list themed on “Best peach dishes” and have peach icecream on that list.
